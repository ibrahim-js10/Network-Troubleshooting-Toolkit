# 🔧 Network Troubleshooting Toolkit

A powerful and user-friendly GUI-based application designed to perform essential network diagnostics and troubleshooting tasks in real time. This toolkit integrates multiple networking utilities into a single interface, making it ideal for beginners, IT support professionals, and system administrators.

---

## 🚀 Features

- 📡 **Ping Test** – Check network connectivity
- 🌐 **IP Configuration** – View detailed network settings
- 🔍 **Traceroute (Tracert)** – Track packet path and delays
- ⚡ **DNS Lookup (NSLookup)** – Resolve domain names
- 🖥️ **System Information** – Display system details
- 🔗 **Port Scanner** – Detect open ports using multithreading
- 📊 **Netstat** – View active network connections
- 🧹 **Clear Console** – Reset output logs
- 💾 **Export Results** – Save diagnostic reports as text files

---

## 🛠️ Technologies Used

- Python  
- Tkinter (GUI Framework)  
- Socket Programming  
- Subprocess Module  
- Multithreading  

---

## 🎯 Objective

To provide a centralized toolkit for performing common network troubleshooting tasks efficiently, while demonstrating practical knowledge of networking concepts used in real-world IT support and DevOps environments.

---

## 💡 Key Highlights

- Interactive and modern GUI design  
- Cross-platform command execution (Windows/Linux)  
- Real-time logging with timestamps  
- Multithreaded port scanning for better performance  
- Export functionality for reports
